# Songbird

Songbird is a midi device built to aid live electronic peformance

Hardware:
- Feather S2
- Featherwing OLED screen
- Featherwing MIDI i/o

Software:
- Circuitpython
- Midi lib
- OLED lib
- Custom music lib
